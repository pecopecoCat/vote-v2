# VOTE v2 テストチェックリスト（スプレッドシート用）

Google スプレッドシート / Excel 用のテストチェックリストです。

## 1ファイル版（おすすめ）

**[VOTE_v2_テストチェックリスト.xlsx](./VOTE_v2_テストチェックリスト.xlsx)** に全シートをまとめています。

| シート | 内容 |
|--------|------|
| 使い方 | 取り込み手順・進め方 |
| チェックリスト | **メイン**（89項目・OK/NG/保留の入力規則付き） |
| テスト環境・凡例 | 端末・KV・アカウント |
| 実施記録 | 回ごとのメタデータ |
| 優先度サマリ | P0/P1/P2 件数・P0 進捗の自動集計 |

### Google スプレッドシートで開く

1. [Google ドライブ](https://drive.google.com) に **VOTE_v2_テストチェックリスト.xlsx** をアップロード
2. ファイルを右クリック → **アプリで開く → Google スプレッドシート**
3. 変換後、シートタブで各シートを切り替えて利用

※ 初回変換時にレイアウトが少しずれる場合は、列幅を手動調整してください。

## Google スプレッドシートへの取り込み（CSV を使う場合）

### 新規スプレッドシートにまとめる場合

1. [Google スプレッドシート](https://sheets.google.com) で **空白のスプレッドシート** を作成
2. **ファイル → インポート → アップロード** で `01_チェックリスト.csv` を選択
3. インポート場所: **新しいシートを作成**
4. 文字コード: **UTF-8**（自動検出されることが多い）
5. 同様に `02`〜`04` も **インポート → 新しいシートを作成** で追加

### 列の使い方（01_チェックリスト）

| 列 | 入力 |
|----|------|
| 結果_iPhone_Safari | `OK` / `NG` / `保留` |
| 結果_Mac_Chrome | 同上 |
| 結果_Firefox_任意 | 任意 |
| メモ | NG の再現手順・スクショURL など |
| データモード | `ローカルのみ` または `KV共有` |

### 結果列をプルダウンにする（任意）

1. `結果_iPhone_Safari` 列（G列）を選択
2. **データ → データの入力規則**
3. 条件: **リストを直接指定** → `OK,NG,保留`
4. 他の結果列にもコピー

### フィルタ・色分け（任意）

- **データ → フィルタを作成** で `優先度 = P0` だけ表示
- 条件付き書式: `NG` のセルを赤、`OK` を緑 など

### P0 進捗の数え方

スプレッドシート関数の例（`結果_Mac_Chrome` が H列、`優先度` が C列の場合）:

```text
=COUNTIFS(C:C,"P0",H:H,"OK")
```
